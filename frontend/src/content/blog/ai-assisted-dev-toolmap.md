# AI辅助研发工具地图

## 概览

这张地图覆盖了 AI 辅助研发的完整流程：

### 🔄 DevOps 流程
- **代码管理**：GitHub Copilot, Cursor
- **CI/CD**：GitHub Actions, Jenkins
- **监控**：Datadog, Prometheus

### 🧠 AI 模型层
- **LLM API**：Claude, GPT-4, Gemini
- **向量数据库**：Chroma, Pinecone
- **RAG**：LangChain, LlamaIndex

### 📝 文档与知识库
- **文档**：Notion, Obsidian
- **知识图谱**：Neo4j
- **搜索**：Elasticsearch

### 🚀 部署与运维
- **容器化**：Docker, Kubernetes
- **Serverless**：Vercel, Netlify
- **云服务**：AWS, GCP, Azure

## 工具链推荐组合

| 场景 | 推荐组合 |
|------|----------|
| 前端开发 | Cursor + Vercel |
| 后端开发 | Copilot + Railway |
| 知识管理 | Obsidian + Chroma |
| 全栈 | Claude + Docker |

## 总结

AI 辅助研发的核心是 **人机协作**：AI 处理重复性工作，人专注创意和决策。
