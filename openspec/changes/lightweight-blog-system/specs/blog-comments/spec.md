## ADDED Requirements

### Requirement: 发表评论
系统必须允许用户对文章发表评论。

#### Scenario: 发表成功评论
- **当** 用户填写评论内容并提交
- **那么** 评论保存到数据库
- **并且** 关联到对应文章
- **并且** 显示"评论发表成功"提示
- **并且** 评论立即显示在评论列表中
- **并且** 响应状态码为201

#### Scenario: 评论内容为空
- **当** 用户提交空评论内容
- **那么** 系统拒绝提交
- **并且** 显示"评论内容不能为空"错误提示
- **并且** 响应状态码为400

#### Scenario: 未登录用户发表评论
- **当** 未登录用户尝试发表评论
- **那么** 系统显示登录提示
- **并且** 引导用户登录后继续操作

### Requirement: 查看评论
系统必须显示文章的所有评论。

#### Scenario: 查看文章评论
- **当** 用户访问文章详情页
- **那么** 页面底部显示该文章的所有评论
- **并且** 评论按时间倒序排列
- **并且** 每条评论显示作者、时间和内容

#### Scenario: 评论分页加载
- **当** 文章评论超过20条
- **那么** 只显示前20条评论
- **并且** 提供"加载更多"按钮
- **并且** 点击加载更多评论

#### Scenario: 评论加载动画
- **当** 评论列表正在加载
- **那么** 显示加载动画
- **并且** 评论逐条淡入显示
- **并且** 每条评论延迟100毫秒显示

### Requirement: 删除评论
系统必须允许用户删除自己的评论，管理员可以删除任何评论。

#### Scenario: 用户删除自己的评论
- **当** 用户点击自己评论的删除按钮
- **那么** 显示确认对话框
- **并且** 确认后评论从数据库移除
- **并且** 评论从页面消失（使用淡出动画）
- **并且** 显示"评论已删除"提示

#### Scenario: 管理员删除任意评论
- **当** 管理员点击任意评论的删除按钮
- **那么** 评论被删除
- **并且** 系统记录删除操作

### Requirement: 评论排序
系统必须支持评论的多种排序方式。

#### Scenario: 按最新评论排序
- **当** 用户选择"最新"排序选项
- **那么** 评论按发布时间降序显示

#### Scenario: 按热门评论排序
- **当** 用户选择"热门"排序选项
- **那么** 评论按点赞数降序显示
- **并且** 点赞数相同则按时间降序

### Requirement: 评论回复
系统必须支持对评论进行回复。

#### Scenario: 回复评论
- **当** 用户点击某条评论的回复按钮
- **那么** 显示回复输入框
- **并且** 输入框下方显示被回复的评论内容预览
- **并且** 提交后回复显示在被回复评论下方
- **并且** 回复缩进显示以体现层级关系

#### Scenario: 嵌套回复显示
- **当** 评论有多层回复
- **那么** 最多显示3层嵌套
- **并且** 超过3层的回复显示"展开更多回复"链接
- **并且** 点击链接展开所有回复

### Requirement: 评论审核
系统必须支持管理员审核评论。

#### Scenario: 新评论待审核
- **当** 用户发表新评论
- **那么** 评论状态为"待审核"
- **并且** 评论不立即公开显示
- **并且** 管理员在后台看到待审核提示

#### Scenario: 管理员批准评论
- **当** 管理员点击"批准"按钮
- **那么** 评论状态变为"已审核"
- **并且** 评论在前台公开显示

#### Scenario: 管理员拒绝评论
- **当** 管理员点击"拒绝"按钮
- **那么** 评论状态变为"已拒绝"
- **并且** 评论不在前台显示
